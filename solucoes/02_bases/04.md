## Problema

4 - Converta 121<sub>3</sub> em base decimal e converta de volta o valor
obtido em base 3. Mostre todos os cálculos.

## Soluções

### Solução 1

**121<sub>3</sub>** para base 10:

1 => 1 x 3<sup>0</sup> = **1**

2 => 2 x 3<sup>1</sup> = **6**

2 => 2 x 3<sup>2</sup> = **9**

1 + 6 + 9 = 16 <sub>10</sub>

**=> 16<sub>10</sub>**

**16<sub>10</sub>** para base 3:

16/3 = 5 + Resto **1**

5/3 = 1 + Resto **2**

**=> 121<sub>6</sub>**

## Problema

3 - Converta 54<sub>6</sub> em base decimal e converta de volta o valor
obtido em base 6. Mostre todos os cálculos.

## Soluções

### Solução 1

**54<sub>6</sub>** para base 10:

4 => 4 x 6<sup>0</sup> = **4**

5 => 5 x 6<sup>1</sup> = **30**

4 + 30 = 34 <sub>10</sub>

**=> 34<sub>10</sub>**

**34<sub>10</sub>** para base 6:

34/6 = 5 + Resto **4**

**=> 54<sub>6</sub>**

## Problema

3 - O que é impresso no ecrã pelo seguinte programa em C?

```c
#include <stdio.h>

void recursiva(int valor) {
    if (valor == 0) {
        printf("Fim!\n");
    } else {
    	printf("%d\n", valor);
        recursiva(valor - 1);
    }
}

int main() {
    int x = 5;
    recursiva(x);
}
```

## Soluções

### Solução 1

É impresso no ecrã 5, 4, 3, 2, 1 e Fim!

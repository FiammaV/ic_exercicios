## Problema

2 - O que é impresso no ecrã pelo seguinte programa em C?

```c
#include <stdio.h>

/* Protótipos das funções. */
int funcao2(int i, int j);
void funcao1(int a, int b);

/* Função principal. */
int main() {
    int x = 1, y = 2, z = 3;
    funcao1(x, y);    
    printf("%d\t%d\t%d\n", x, y, z);
}

/* Corpos das funções. */
int funcao2(int i, int j) {
    int y = i + 1;
    int z = j + 1;
    printf("%d\t%d\n", y, z);
    return y + z;
}

void funcao1(int a, int b) {
    int x = -1, y = -2, z = -3;
    x += funcao2(a, b);
    printf("%d\t%d\t%d\n", x, y, z);
}
```

## Soluções

### Solução 1

São impressos no ecrã os valores 2, 3, 4, -2, -3, 1, 2 e 3.

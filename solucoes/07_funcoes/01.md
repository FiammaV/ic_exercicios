## Problema

1 - O que é impresso no ecrã pelo seguinte programa em C?

```c
#include <stdio.h>

void func(unsigned char * y) {
    unsigned char x = ++(*y) - 6;
    printf("%u ", x);
}

int main() {
    unsigned char x = 4;
    func(&x);    
    printf("%u\n", x);
}
```

## Soluções

### Solução 1

São impressos no ecrã os valores 255 e 5.
